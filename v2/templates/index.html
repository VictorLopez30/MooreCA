Descripción de las métricas

Entropía de Shannon

Mide la incertidumbre de la distribución de los pixeles de una imagen. Las imágenes tienen valores RGB y la cantidad de valores de un píxel son 256, por lo tanto, la entropía cercana a 8 determina una alta incertidumbre.

Chi-cuadrada

Compara el histograma observado contra una distribución ideal uniforme, es una medición que se hace para reforzar la incertidumbre y no se considera en las normas NIST.

El valor ideal de esta métrica para una imagen RGB es de 255, donde valores más bajos suelen indicar menos sesgo en las intensidades; valores altos sugieren concentración en ciertos tonos.

Correlación

Evalúa la relación entre píxeles vecinos. En una imagen natural suele ser alta porque los píxeles cercanos se parecen.

En una imagen cifrada se busca que esté cercano a 0, indicando baja dependencia espacial.

Histograma RGB

Muestra cuántas veces aparece cada valor de intensidad en un valor de pixel en los canales rojo, verde y azul.

Sirve para detectar picos, saturaciones o tendencias visibles en la imagen.

Cómo funciona el cifrado

1. Imagen de entrada

La imagen se convierte a texto claro. El cifrado trabaja sobre los bytes de píxeles, no sobre el archivo directamente.

2. Llaves de sesión

Se obtiene un conjunto de datos de manera secreta de longitud de 32 bytes utilizando la herramienta salt. Con HKDF-SHA256 se derivan llaves para permutación, autómata y sesión.

La interfaz permite usar una sesión independiente por lenguaje o una sesión compartida para los tres.

3. Permutación

En cada ronda se elige una de 128 permutaciones posibles, las cuales están basadas en AES. Esta reorganiza espacialmente los píxeles mediante desplazamientos por filas y columnas.

4. Autómata de Moore

Después de permutar, se aplica una generación del autómata celular con vecindad de Moore y aritmética módulo 257.

Cada ronda usa coeficientes derivados de la llave, por lo que la evolución cambia ronda a ronda.

5. Rondas

Una ronda equivale a una permutación más una generación del autómata. Aumentar rondas incrementa la difusión, pero también el tiempo de cómputo.

6. Descifrado

El sistema guarda dos estados finales y un archivo de sesión. Con esos datos se revierte el autómata y se recupera la imagen original.

Las métricas ayudan a observar comportamiento estadístico, pero no sustituyen un análisis criptográfico formal. En esta interfaz se usan como apoyo para comparar difusión, uniformidad y recuperación entre las implementaciones en C, Java y C#.
